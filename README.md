# Analizador Lexico y Sintactico - Pila

Proyecto de construccion de un analizador lexico y sintactico para un lenguaje de operaciones de Pila, desarrollado con ANTLR4 y Node.js

## Requisitos previos

Para ejecutar este proyecto es necesario tener instalado Node.js en el sistema

## Instalacion

1. Clonar este repositorio
2. Abrir una terminal en la carpeta del analizador ya que es donde se encuentra el archivo index.js
3. Ejecutar el siguiente comando para instalar las dependencias:

   npm install

## Ejecucion

El analizador recibe como argumento la ruta de un archivo de texto con el codigo fuente. Para probar la aplicacion con los ejemplos provistos, utilice el siguiente formato en la terminal:

   node index.js correcto_1.txt

Se puede reemplazar "correcto_1.txt" por cualquiera de los otros archivos de prueba incluidos en el repositorio:
- correcto_2.txt
- incorrecto_1.txt
- incorrecto_2.txt
